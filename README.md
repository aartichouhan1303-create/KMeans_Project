# 🌍 Country Clustering Dashboard

An interactive Machine Learning dashboard that clusters countries based on socio-economic and health indicators using the KMeans algorithm.

## Features

- Interactive Dashboard
- Country Cluster Prediction
- Data Visualization
- Dataset Preview
- Country Search
- Modern Streamlit UI

## Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- Plotly
- Joblib

## Installation

```bash
git clone https://github.com/yourusername/Country-Clustering-Dashboard.git

cd Country-Clustering-Dashboard

pip install -r requirements.txt

streamlit run app.py
```

## Dataset

Country Development Dataset

## Model

KMeans Clustering
